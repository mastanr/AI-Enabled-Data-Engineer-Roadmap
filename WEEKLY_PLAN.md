# 24-Week Practical Study Plan

## Month 1 — Python + SQL Refresh

### Week 1
- Setup Python, VS Code, GitHub, Jupyter
- Learn Python variables, lists, dictionaries, loops
- Practice 10 small Python problems

### Week 2
- Learn functions, files, error handling
- Start NumPy and pandas
- Read and clean CSV files

### Week 3
- SQL joins, CTEs, GROUP BY
- Practice 15 SQL queries
- Build source-to-stage table logic

### Week 4
- SQL window functions
- Incremental load pattern
- Watermark table design
- Mini project: CSV → stage → final table design
