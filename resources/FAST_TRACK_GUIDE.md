# Fast Track Guide

If your goal is Microsoft/Product-company Data Engineering with AI skills, follow this fast track.

## Daily routine: 60 minutes

```text
15 min — Python / SQL practice
15 min — Concept learning
20 min — Hands-on notebook or mini pipeline
10 min — Notes + GitHub update
```

## Weekly output

Every week, produce one visible artifact:

- A notebook
- A SQL script
- A mini pipeline design
- A README note
- A small AI/RAG demo
- An architecture diagram

## Avoid this mistake

Do not spend 3 months only watching videos. Build small things from week 1.

## Best first 4 projects

1. CSV Data Quality Checker
2. SQL Incremental Load Framework
3. ML Model from Gold Table
4. RAG App over Technical Documents
