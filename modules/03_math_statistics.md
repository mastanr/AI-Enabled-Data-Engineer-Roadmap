# Module 03 — Math and Statistics for AI

## Goal
Learn enough math and statistics to understand ML and AI.

## Learn
- Boolean logic
- Truth tables
- Sets
- Relations
- Functions
- Probability
- Mean, median, mode
- Variance and standard deviation
- Correlation
- Confusion matrix
- Precision and recall
