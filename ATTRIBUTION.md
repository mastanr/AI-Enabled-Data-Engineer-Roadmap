# Attribution and External Resource Usage

This repository is a learning roadmap and portfolio guide created by the repository author.

## Important Usage Statement

This repository does **not** copy, host, or redistribute full course materials from MIT OpenCourseWare, Microsoft Learn, Google, Kaggle, Databricks, OpenAI, or any other external provider.

This repository only includes:

- Original roadmap structure
- Original study guidance
- Original notes created by the repository author
- Links to external learning resources
- Short descriptions of what to study from each resource

All external resources remain the property of their respective owners.

---

## MIT OpenCourseWare Attribution

Some topics in this roadmap may recommend MIT OpenCourseWare as an external learning resource.

MIT OpenCourseWare materials are available under their own license terms, commonly Creative Commons Attribution-NonCommercial-ShareAlike unless otherwise marked.

When using MIT OpenCourseWare materials:

- Give proper credit to MIT OpenCourseWare and the course author/instructor when required.
- Link back to the original MIT OpenCourseWare course page.
- Do not use MIT names, logos, seals, or branding in a way that suggests endorsement.
- Do not copy entire lectures, slides, videos, assignments, exams, or solution files into this repository.
- Do not commercialize MIT OpenCourseWare materials or derivative works based on them.
- Check the license notice on the specific MIT OCW course page before reusing any material.

Official MIT OCW terms:

- https://ocw.mit.edu/pages/privacy-and-terms-of-use/
- https://mitocw.zendesk.com/hc/en-us/articles/4414774353051-What-are-the-requirements-of-use-for-MIT-OpenCourseWare
- https://mitocw.zendesk.com/hc/en-us/articles/4414782785819-How-does-MIT-define-non-commercial-use

## Safe Attribution Format

Use this format when adding notes based on an external course:

```markdown
## References

This note was created in my own words after studying the following public learning resource:

- Course/resource: [Resource Title](RESOURCE_URL)
- Provider: Provider Name
- Instructor/Author: Instructor Name, if available
- License/terms: See the original resource page

This repository does not host or redistribute the original course materials.
```
