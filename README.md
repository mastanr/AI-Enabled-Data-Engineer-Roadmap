# 🚀 AI-Enabled Data Engineer Roadmap
### Learn Python, SQL, Data Engineering, Machine Learning, GenAI, RAG, and Agentic AI from scratch

Welcome to the **AI-Enabled Data Engineer Roadmap**.

This repository is designed for **Data Engineers, BI Developers, SQL Developers, Azure Developers, and beginners** who want to move into:

- Data Engineering
- Machine Learning
- Generative AI
- RAG Applications
- Agentic AI
- Azure + Databricks AI projects

The goal is simple:

> Learn AI/ML practically from a Data Engineering point of view.

## 🎯 Who Is This Roadmap For?

This roadmap is useful if you are:

- A Data Engineer learning AI/ML
- A SQL Developer moving into Python and ML

## 🧭 Learning Path Overview

The roadmap is divided into modules from beginner to advanced.

You do **not** need to master everything at once.

Follow this order:

```text
Python
→ SQL
→ Data Engineering
→ Math + Statistics
→ Machine Learning
→ Databricks + MLflow
→ GenAI
→ RAG
→ Agentic AI
→ Portfolio Projects
```

# Module 0 - Before You Start

Before learning AI and ML, set up your system properly.

## What You Need

- Laptop or desktop
- Internet connection
- Python installed
- VS Code installed
- Git installed

## Learn

- How to use GitHub
- How to create folders and notebooks
- How to run Python code
- How to use VS Code
- How to write clean notes

## Recommended Links

- [Python Official Download](https://www.python.org/downloads/)
- [VS Code Download](https://code.visualstudio.com/)
- [Git Download](https://git-scm.com/downloads)
- [GitHub Docs](https://docs.github.com/en)
- [Jupyter Notebook Docs](https://docs.jupyter.org/en/latest/)

## Outcome

By the end of this module, you should be able to:

- Install Python
- Run Python code
- Create a GitHub repo
- Create a notebook

## Learn Statistics

- Mean
- Median
- Mode
- Variance
- Standard deviation
- Probability
- Distribution
- Correlation
- Hypothesis testing
- Confusion matrix
- Precision
- Recall
- F1 score

## Recommended Links

- [Khan Academy Statistics and Probability](https://www.khanacademy.org/math/statistics-probability)

| Concept | Data Engineering / AI Use |
|---|---|

| Sets | Joins and duplicate checks |
| Functions | Data transformation |

| Variance | Data spread |
| Correlation | Feature relationship |
| Precision / Recall | Model evaluation |
